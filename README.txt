## Project Overview

This internal web application serves the research team's needs by providing comprehensive data management tools. It includes CRUD functionalities for various tables, real-time updates, and user-friendly interfaces.

## Key Features

- **Data Management**: Manage various data tables, including `history_lateralizado`.
- **Real-time Updates**: Seamless updates using jQuery/AJAX.
- **User-friendly UI**: Includes pop-up logic, edit-in-cell functionality, and dropdown lists.
- **Error Handling**: Robust error handling for database operations and user interactions.

## Technologies and Frameworks

- Backend: Python, Flask, SQLAlchemy
- Frontend: jQuery, AJAX, HTML, CSS
- Database: MySQL
